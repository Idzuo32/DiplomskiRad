A 3D endless runner game built in Unity as a final exam project for the Bachelor of Game Development program at Metropolitan University (2025).

## Background

This project was developed as the final exam submission for the Bachelor of Game Development degree at Metropolitan University, Belgrade. It demonstrates core gameplay programming skills including procedural content generation, difficulty scaling, and score tracking.

## Tech Stack

| Tool | Version |
|------|---------|
| Unity | 2022 LTS+ |
| Language | C# |
| Render Pipeline | Universal Render Pipeline (URP) |
| Input System | Unity Input System |
| TextMesh Pro | Included |

## Features

- **Procedural Level Generation** — terrain and obstacles are generated at runtime, ensuring a unique run every time
- **Progressive Difficulty** — speed and obstacle density scale over time as the player survives longer
- **Scoring System** — tracks distance and in-run score with persistent high score
- **C# Gameplay Programming** — clean, component-based architecture following Unity best practices

## Getting Started

### Prerequisites

- [Unity Hub](https://unity.com/download)
- Unity Editor **2022.3 LTS** or newer
- Git (to clone the repository)

### Running the Project

```bash
git clone https://github.com/markostankovic/endless-runner.git
cd endless-runner
```

1. Open **Unity Hub**
2. Click **Open** → **Add project from disk**
3. Select the cloned project folder
4. Unity Hub will automatically detect the correct editor version — install it if prompted
5. Once the project loads, open `Assets/_Project/Scenes/MainMenu.unity`
6. Press **Play** in the Unity Editor to run the game

> No additional setup or package installation is required. All dependencies are managed via the Unity Package Manager and are included in the `Packages/` directory.
